Description

Select Multiple Invoices then Print

Go to the Invoicing -> Select any number of Invoices you wish to generate, and then click Action -> Multiple Separate Invoices.


Print Report Separate with Invoice & without Payment Invoice

You can choose to select between Invoice and Invoice without Payment options and click on Download button to generate.


Download Separate PDF in zip by Single Click

By clicking on Download, it again opens a window where you can see the Downloadable ZipFile, where you can click and download on your local system.


Generated Separate PDF Files in Zip

You can see the Separate PDF Files generated for each record you select, and you can extract these files anywhere.


Print Separate Report with Choose Invoice Option

Similarly, you can again select the Invoices and click on Action -> Multiple Separate Invoices and can choose Invoice option to generate ZipFile.


Download Generated Separate Invoice PDF Report in zip Format

The ZipFile is generated containing all the Separated PDF Files in it, which you can extract anywhere on your system.


Separate PDF Report for Vendor Bills

Similarly, Go the Vendors -> Bills and you can select any number of Bills to be generated as Separate PDF Files.


Print Separate Bills using this Menu above as According Invoices

Select any number of bills and then click on Action -> Multiple Separate Invoices and follow the same process again, and you can generate Separate PDF Files of Bills With & Without Payment in one ZipFile.
